📘 Coaching Management System

Complete documentation for administrators, staff, and developers

🧾 Overview

This system helps manage all aspects of a coaching center efficiently including students, teachers, attendance, payment, Exam, SMS messaging, and teaching assignments. It also supports report generation in PDF format.

📦 Key Modules

🎓 Student Management

  • Create, update, delete students
  • View active/inactive students
  • Student profile view

🧑‍🏫 Teacher Management

  • Manage teacher profiles
  • Assign classes
  • Deactivate/reactivate teachers

📅 Attendance

  • Mark daily attendance
  • Generate attendance reports
  • View student attendance history
  • Automatically send SMS to Absent Student

💳 Payment & Credit

  • Pay monthly fees
  • Track due/paid months
  • View credit reports

📤 SMS Messaging

  • Send bulk or individual messages
  • Reset SMS counter

📚 Teaching Assignments

  • Assign subjects to teachers
  • Manage class-subject combinations

📝 Exam Results

  • Create and update exam results
  • View and manage exam records
  • Generate detailed performance reports
  • Send exam result through SMS

🏆 Exam Positions

  • Generate student rankings
  • Track yearly exam positions
  • Track monthly exam positions
  • View last exam position for each student

📑 PDF Reports

  • Generate PDF for all student list
  • Generate PDF for all teacher list
  • Export yearly and monthly performance reports As PDF

🚀 How to Use

  1. Login with your credentials.
  2. Navigate to the desired module (Student, Teacher, etc.) from the dashboard.
  3. Perform create/update/delete operations as required.
  4. Use PDF options to export reports if needed.
  5. Use the SMS module to communicate with students/parents.

💻 System Requirements

  • Python 3.9+ with Django Framework
  • MySQL or PostgreSQL database
  • Custom CSS for frontend styling
  • Internet connection for SMS API (e.g., Twilio or local provider)
